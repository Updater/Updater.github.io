Grid

Updater uses a 12-column fluid grid that is built on Bootstrap 3. The size of the container responds to the user’s viewport size. The size of the elements within containers is defined as a percentage of the parent element.

As screen sizes become smaller, we can use classes to cause side-by-side elements to either shrink in percentage or stack vertically. We can also hide or show elements depending on screen size. It's our responsivility to ensure end-users on all screen sizes have an equally enjoyable experience on our site.

We use SCSS to mixin predefined styles from Bootstrap into our styles. Jump over here for more details on Bootstrap grids.
